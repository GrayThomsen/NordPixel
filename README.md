# NordPixel – Teknologiforståelse

Next.js hjemmeside for NordPixel med fokus på teknologiforståelse, WebLab og skoleklare forløb.

## Indhold
- Hovedside (`/`)
- Forløbsside (`/courses`)
- Bookingflow (`/courses/booking`)
- Kontakt (`/contact`)
- WebLab-editor (`/editor`)

## Tech Stack
- Next.js 15 (App Router)
- React 18 + TypeScript
- Tailwind CSS 4 + custom CSS tokens

## Kom i Gang
Dette projekt bruger kun Node.js/npm (ingen Python eller `.venv` er nødvendig).

1. `npm install`
2. `npm run dev`
3. Åbn `http://localhost:3000`

## Kommandoer
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run clean`
