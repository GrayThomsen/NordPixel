# NordPixel – Teknologiforståelse

Minimal hjemmeside for NordPixel – fokuseret på undervisning i teknologiforståelse for folkeskoler.

## Indhold
- Hovedside med information om undervisning og booking (`/`)
- Om-side (`/about`)
- Direkte email-kontakt (ingen formular)
- LinkedIn-links (virksomhed + personlig)

## Tech Stack
- Vite
- React 18 + TypeScript
- Tailwind CSS + custom CSS
- React Router v6

## Kom i Gang
Dette projekt bruger kun Node.js/npm (ingen Python eller `.venv` er nødvendig).

1. `npm install`
2. `npm run dev`
3. Åbn `http://localhost:5173`

## Kommandoer
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
